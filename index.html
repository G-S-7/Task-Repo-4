Introduction to HTML

HTML (Hypertext Maekup Language) is used to create documents on the World Wide Web. It is simply a collection of certain keywords called "Tags" that are helpful in writing the document to be displayed usinga browser on Internet. HTML was developed by Tim Berners-Lee in 1992.
